# {{VITE_APP_NAME}} - Image to SVG Converter

> Convert raster images to scalable vector graphics with AI-powered vectorization.

## Overview

{{VITE_APP_NAME}} is a web application that converts raster images (PNG, JPG, WEBP) into clean, scalable SVG files. Upload your images via drag-and-drop and get vector graphics suitable for logos, icons, illustrations, and print. The app features SVG and PDF output, a credit-based download system, and RevenueCat subscription support. Especially useful for converting AI-generated designs into production-ready SVGs.

## Core Features

- Drag-and-drop image upload (PNG, JPG, WEBP)
- AI-powered SVG conversion
- PDF export support
- Credit-based download system (1 credit per download)
- RevenueCat subscription integration
- Batch processing support
